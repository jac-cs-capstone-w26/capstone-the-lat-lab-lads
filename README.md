# JAC-CS Capstone Project Winter 2026

## Finishing Milestone 1

- Create a branch called `main` from the **first commit hash** on the `instructions` branch**[^1].
    - See [the Connected Objects Lab 5](https://john-abbott-college.github.io/6P3-Notes/labs/coursework-feedback/index.html)
    instructions for notes on how to create a branch from a particular commit hash.
- Create a pull request from your `m1` branch to the new `main` branch.
- Review the `m1` changes:
    - Make sure all team members have answered relevant questions
    - Put *comments* into the pull requests if you have any questions or extra comments
- When you are ready, click the **merge and rebase** option to merge `m1` and incorporate
  all its changes into main.

[^1]: if you have already created a `main` branch, you don't need to recreate it.

## Starting Milestone 2

- Create a branch called `m2` from the [HEAD](https://git-scm.com/docs/user-manual.html#def_head) of the `instructions` branch.
- The rest of the instructions for milestone 2 can be found on the course websites for
Connected Objects and App Dev III
